commands

you can compile and run the code with the following command:

make run

you can run the tests with the following command:

make run-tests

structure

The code is structured in 3 files:

LidDrivenCavitySolver

This file contains the main function. It reads the input parameters and creates an instance of the LidDrivenCavity class. It then calls the Initialise and Integrate methods of the LidDrivenCavity class. It also writes the initial and final solutions to files.

LidDrivenCavity

This file contains the implementation of the LidDrivenCavity class. This objects stores the simulation parameters and the solution and has the methos initialise, integrate and write the solution to a file.

SolverCG

This file contains the implementation of the SolverCG class. This class implements the Conjugate Gradient method to solve a linear system of equations.
